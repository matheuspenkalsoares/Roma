# Guia rápido para subir no GitHub

## Opção 1 — Pelo site do GitHub

1. Acesse github.com e crie um novo repositório.
2. Nome recomendado: `MPENKAL-TRAINING`.
3. Marque como público se quiser usar GitHub Pages gratuito.
4. Clique em **Add file > Upload files**.
5. Arraste todos os arquivos e pastas deste projeto.
6. Clique em **Commit changes**.
7. Vá em **Settings > Pages**.
8. Em **Branch**, selecione `main` e `/root`.
9. Salve.

## Opção 2 — Pelo terminal

```bash
git init
git add .
git commit -m "Primeira versão do site MPENKAL TRAINING"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/MPENKAL-TRAINING.git
git push -u origin main
```

Depois ative o GitHub Pages em **Settings > Pages**.
